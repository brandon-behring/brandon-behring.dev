## Root-cause finding (de-risking)
The lab graph's junk titles trace to **research-kb storing `title = arxiv_id`, `authors = []`,
`year = <arxiv-prefix>`** (`metadata_source: "filename"` — RL papers were auto-ingested by
filename). So research-kb **cannot** supply real titles, and densifying *from* it would import
more junk. Enrichment therefore pulls from the **arXiv API** (offline fallback:
`~/Claude/rl_and_control/references/paper_index.md`). Graph densification is now **blocked-on**
the research-kb metadata-extraction fix.
